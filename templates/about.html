What is PALT?

PALT or Packet Analysis Learning Tool is a web application learning tool that provides real-time capture of network traffic packets. These captured packets are then broken down and simplified, to allow for a comprehensive learning experience.


PALT was created as a tool for aspiring Computer Science students who are curious about the inner workings of network traffic.
