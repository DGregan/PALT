Packet Analyzing Learning Tool

Overview


Learning about Network Traffic for the first time can be a challenging experience for a Computer networking novice or student. Packet Analysing Learning Tool (PALT) will attempt to erode the arduous task of mastering the fundamentals of Networking Traffic.


This will be accomplished by providing an environment where users can learn about network traffic through the capture of live traffic and the dissecting of packets travelling across the network. The captured network packets will then be inspected and return the analysed contents to the users.
